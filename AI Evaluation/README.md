AI-Powered Daily Time Tracking & Analytics Dashboard
Project Description

This project is a Time Tracking Web Application designed to help users log daily activities in minutes and visually analyze how their 24 hours are spent. The focus is on clean architecture, responsive UI, and AI-assisted design.

Current Progress:
Figma designs for Login, Dashboard, Tasks, and Calendar pages have been completed. Functional implementation is yet to be done.


Tech Stack

Frontend: HTML, CSS, JavaScript (DOM manipulation)

Backend/Database: Firebase (Authentication + Firestore/Realtime DB) – planned

UI/UX Design: Figma

AI Tools Used: Used AI for initial design and layout ideas

Features (Planned)

User Authentication: Firebase Authentication (Email/Google)

Activity Logging: Add, edit, delete activities with duration and optional category; validate total ≤ 1440 minutes per day

Analytics Dashboard: Date-based activity breakdown, visualizations (pie/bar/timeline), “No data available” state

Responsive Design: Works on mobile, tablet, and desktop

AI Integration: AI-assisted design and idea generation

Current Status: Only the design mockups are completed. Functional implementation is pending.
